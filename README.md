# Banking & FinTech Analytics Project

## Project Overview

This project is an end-to-end Banking and FinTech Analytics solution developed using Python, SQL, Power BI, and Google Colab.

The project focuses on banking transaction analysis, fraud detection, customer insights, and business intelligence dashboard creation.

---

## Technologies Used

* Python
* SQL
* Power BI
* Pandas
* NumPy
* Matplotlib
* Google Colab

---

## Project Features

* Banking Transaction Analysis
* Fraud Detection Analysis
* SQL Business Queries
* Interactive Power BI Dashboard
* Customer Spending Insights
* Financial Trend Analysis

---

## Dashboard KPIs

* Total Transactions
* Fraud Transactions
* Average Transaction Amount
* Monthly Transaction Trends
